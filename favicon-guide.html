How to Add a Favicon

Follow these steps to add a favicon to your website:

  1. Create your favicon:
    • Visit favicon.io
    • Use their text-to-favicon generator with the letter "A"
    • Choose colors matching your brand (primary: #0066cc)
    • Download the generated files
  2. Alternative - Use RealFaviconGenerator:
    • Visit realfavicongenerator.net
    • Upload an image or logo
    • Generate all sizes for different platforms
  3. Add files to your project:
    • Create a folder named images or assets in your root directory
    • Place all favicon files there
  4. Add to your HTML <head> section: 
    <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/images/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/images/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/images/apple-touch-icon.png">

Quick SVG Favicon (Modern Browsers)

For a quick solution, you can create a simple SVG favicon and add it to your HTML:

<link rel="icon" type="image/svg+xml" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><rect width='100' height='100' fill='%230066cc'/><text y='75' font-size='80' fill='white' font-family='sans-serif' font-weight='bold'>A</text></svg>">

Note: Add the favicon link tags to all HTML files in your project (index.html, servicios.html, sobre-nosotros.html, contacto.html).
